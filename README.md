# MoviesListSample
The app shall fetch the movies data from The Open Movie Database (OMDb) REST API for iOS written in Swift.

![Simulator Screen Shot - iPhone 13 Pro - 2023-02-11 at 14 25 54](https://user-images.githubusercontent.com/3304617/218254083-b229d711-f71e-4f94-8781-7fccd9ae62c0.png)
# Demo
Do pod install in your console and run the project. To install CocoaPods, run sudo gem install cocoapods in your console.
# Basic usage
Typing title of movies that you want to find on search bar.
